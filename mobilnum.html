Form mobile number Validation

by using regular expression mobile number can be validate

\d for digit instead of writing [0-9], \w for word instead of writing [a-z]

^ this used for start point check, * this used for end point check

for ex. /^[7-9]\d{9}/, [7-9] this for first digit start with 7-9 , here \d for digits , here {9} range of digits

Form email Validation

by using regular expression email can be validate

eg. /^([a-zA-Z0-9\.-]+)@([a-zA-Z0-9-]+).([a-z]{2,9})(.[a-z]{2,8})?$/
